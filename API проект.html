Модель Post:

api/posts/ :
GET - получение списка всех постов (поле author - уникальный username)
POST - создание одного поста

api/post/pk/ :
GET - информация о посте и все комменты этого поста в списке по id
PUT - полное обновление поста
PATCH - частичное редактирование поста
DELETE - удаление поста

Модель User:

api/users/ :
GET - получение списка всех пользователей
POST - создание пользователя

api/user/pk/ :
GET - информация о пользователе и список его постов по id
PUT, PATCH, DELETE - изменение пользователя

api/user/pk/posts/ :
GET - спискок всех постов пользователя

api/user/pk/comments/ :
GET - список всех комментов пользователя

Модель Comments:

api/comments/ :
GET - получение списка всех комментов (поле author - уникальный username)
POST - создание коммента

api/comment/pk/ :
GET, PUT, PATCH, DELETE - получение, редактирование, удаление коммента
